# 其余问题

## 如果不是中文的或者报错了

可能你操作有问题。

可能你没有安装补丁。

可能翻译版本有问题。

tips1：如果游戏中出现□□□无法显示，那么可能game文件夹下的字体文件可能有问题。

tips2：如果游戏在使用最新的翻译报错后，删除所有rpyc文件并重新开档，如果还是出现了报错，请反馈给我。

tips3：如果出现漏翻什么的，反馈给我哪里出现漏翻，具体做了什么出现漏翻。
